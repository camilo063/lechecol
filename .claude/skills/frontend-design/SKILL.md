# frontend-design

Create distinctive, production-grade frontend interfaces with high design quality.

## Strategic Approach
Before coding, identify:
- Purpose and intended audience
- Desired tone (industrial, premium, warm, technical...)
- Technical constraints
- ONE memorable differentiator that makes the interface unforgettable

## Aesthetic Execution

**Avoid overused choices:**
- Standard fonts (Arial, Inter, plain Roboto)
- Predictable purple gradients
- Cookie-cutter card layouts
- Generic hero + features grid

**Instead:**
- Select characterful typography: pair distinctive display fonts with refined body typefaces
- Develop cohesive color schemes with a dominant color and sharp, unexpected accents
- Implement thoughtful motion: CSS animations + scroll-triggering, not just fade-in
- Create spatial compositions using asymmetry, overlap, and generous negative space
- Apply atmospheric backgrounds, textures, grain overlays matching the overall vision

## Implementation Philosophy
Match code complexity to aesthetic vision:
- Maximalist designs warrant elaborate animations
- Minimalist approaches demand precision in spacing and typography refinement

Commit fully to a distinctive vision. Never default to generic solutions.
